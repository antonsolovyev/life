http://cwaylife.appspot.com/

##Notes

1. To run locally either:

    `mvn jetty:run`
    
    or
    
    `mvn appengine:devserver`

2. Publish to Google app engine:

    `mvn appengine:update`

3. The URL at Google is http://cwaylife.appspot.com/
